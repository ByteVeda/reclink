# Interactive Playground

Try reclink's functions directly in your browser. Upload CSV files, compare strings with different metrics, build preprocessing pipelines, and run full record linkage workflows — no installation required.

<PlaygroundEmbed height="800px" />

## What you can do

- **String Metrics** — Compare two strings with any of 20+ metrics, see scores side-by-side
- **Phonetic Algorithms** — Encode strings with Soundex, Metaphone, Double Metaphone, and more
- **Preprocessing** — Build and test preprocessing pipelines with live preview
- **Batch Operations** — Upload a CSV and run batch matching, cdist heatmaps, or pairwise similarity
- **Index Structures** — Build BK-trees, VP-trees, and N-gram indexes, then query them
- **Record Linkage** — Configure and run a full deduplication or linkage pipeline on your data

## Running locally

You can also run the playground on your machine:

```bash
cd playground
pip install -r requirements.txt
streamlit run Home.py
```

The playground opens at `http://localhost:8501`.
