# Interactive Playground

Try reclink's functions directly in your browser. All computation runs client-side via WebAssembly — no server, no installation required.

## Available Tools

| Tool | Description |
|------|-------------|
| [**String Metrics**](string-metrics) | Compare two strings with 21 similarity and distance metrics |
| [**Phonetic Algorithms**](phonetic) | Encode strings with 10 phonetic algorithms |
| [**Preprocessing**](preprocessing) | Build and test text preprocessing pipelines |
| [**Batch Operations**](batch-operations) | Match strings in batch and compute similarity matrices |
| [**Index Structures**](indexes) | Build and query BK-tree, VP-tree, N-gram, and MinHash indexes |
